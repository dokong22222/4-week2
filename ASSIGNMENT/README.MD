# 과제 안내

이번 주차 과제는 API 키를 다루는 방법과 깃허브 협업 과정을 익히는 것입니다. 아래의 단계를 따라 과제를 진행해 주세요.

## 과제 진행 순서

### 1단계: 본인 폴더 및 환경변수 출력 스크립트 생성
- `ASSIGNMENT/` 디렉토리 아래에 **본인 이름(또는 닉네임)**으로 폴더를 생성합니다.
- 생성한 폴더 내부에 `.env` 파일에 저장된 환경변수 키를 읽어와 출력하는 `.py` 파일을 만듭니다.
  - 예: `ASSIGNMENT/kjb/main.py`

### 2단계: 개별 README 작성 및 캡쳐 업로드
- 본인 폴더 안에 **README.md** 파일을 생성합니다.
- README.md에는 다음 내용을 포함합니다:
  - 간단한 인사말 또는 학습 내용 정리
  - **증빙 캡쳐본**: `.env`(키값은 가리기), `.py` 코드, 그리고 **터미널 출력 결과물**이 모두 한 화면에 나오도록 캡쳐하여 첨부합니다.
- 작성이 완료되면 본인의 레포지토리에 `push`합니다.

### 3단계: Pull Request 요청
- `HateSlop` 메인 레포지토리로 **Pull Request(PR)**를 보냅니다.
- PR 제목 형식: `[Week2] 본인이름`

---

## 주의사항
- `.env` 파일은 절대 깃허브에 직접 올라가지 않도록 `.gitignore` 설정을 확인하세요.
- 캡쳐 사진에서 실제 API 키값은 반드시 가리거나 모자이크 처리해 주세요.
