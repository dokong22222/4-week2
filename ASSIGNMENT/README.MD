# 과제 안내

## 과제 내용

**본인 폴더에 README 파일을 만들고, 원하는 내용을 작성하세요.**

### 진행 방법

1. **폴더 위치**  
   `ASSIGNMENT/본인폴더명/` 아래에 작업합니다.  
   (본인 폴더가 없다면 `ASSIGNMENT/` 아래에 본인 폴더를 만든 뒤 진행합니다.)

2. **README 파일 생성**  
   해당 폴더 안에 **README.md** 파일을 생성합니다.  
   - 경로 예: `ASSIGNMENT/본인깃허브명/README.md`  
   - 파일명은 `README.md`로 통일합니다. (대소문자 포함)

3. **내용 작성**  
   README.md에는 **자유롭게 원하는 내용**을 적습니다.  
   예시:
   - 자기소개
   - 이번 주차 학습 정리
   - 과제 하면서 느낀 점
   - 사용한 API나 코드에 대한 간단 설명
   - 마크다운으로 제목, 목록, 코드 블록 등 활용

### 정리

| 항목 | 설명 |
|------|------|
| 위치 | `ASSIGNMENT/본인폴더명/README.md` |
| 파일명 | `README.md` |
| 내용 | 자유 작성 (원하는 내용) |

위 조건만 지키면 형식이나 분량에 대한 별도 제한은 없습니다.
